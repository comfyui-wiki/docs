## 连接节点的链接

在ComfyUI的术语中，节点之间的线或曲线被称为**_链接_**。它们也被称为**_连接_**或线路。链接可以以多种方式显示，例如直线（默认）、样条曲线或完全隐藏。链接的显示是从ComfyUI的**设置**窗口和ComfyUI主窗口右下角的显示工具栏上的**切换链接可见性**按钮控制的。

链接的显示是至关重要的。根据情况，可能需要看到所有链接。特别是在学习、分享或只是理解工作流时，链接的可见性使用户能够通过图形跟踪数据的流向。对于不打算修改的打包工作流，可能有必要隐藏链接以减少混乱。

### 重定向节点

如果图形结构的可读性很重要，那么链接线路可以使用一个名为**重定向**的小节点在图形的2D空间中手动路由。其目的是为了确保可见性，定位链接线路的开始和/或结束点。我们可以设计一个工作流，使得链接线路不经过节点，不与其他链接线路交叉等等。

## 颜色编码

节点属性的数据类型通过输入/输出端口和链接连接线的颜色编码来指示。我们总是可以通过颜色来判断哪些输入和输出可以连接到一起。端口只能连接到同色端口。

常见的数据类型：

| 数据类型                          | 颜色                                                                                                              |
|------------------------------------|--------------------------------------------------------------------------------------------------------------------|
| 扩散模型                          | 薰衣草色                                                                                                           |
| CLIP模型                          | 黄色                                                                                                             |
| VAE模型                           | 玫瑰色                                                                                                               |
| 条件                              | 橙色                                                                                                             |
| 潜在图像                          | 粉红色                                                                                                               |
| 像素图像                          | 蓝色                                                                                                               |
| 遮罩                              | 绿色                                                                                                              |
| 数字（整数或浮点数）                | 浅绿色                                                                                                             |
