<Tabs>
<Tab title="Windows/Linux">
| 快捷键                           | 命令                                                                                                        |
|------------------------------------|--------------------------------------------------------------------------------------------------------------------|
| Ctrl + Enter                       | 将当前图形排队以便生成                                                                              |
| Ctrl + Shift + Enter               | 将当前图形作为第一个排队以便生成                                                                     |
| Ctrl + Z / Ctrl + Y                | 撤销/重做                                                                                                          |
| Ctrl + S                           | 保存工作流                                                                                                      |
| Ctrl + O                           | 加载工作流                                                                                                      |
| Ctrl + A                           | 选择所有节点                                                                                                   |
| Alt + C                            | 收起/展开选中的节点                                                                                 |
| Ctrl + M                           | 静音/取消静音选中的节点                                                                                         |
| Ctrl + B                           | 绕过选中的节点（类似于从图形中移除了该节点，并重新连接了线路）            |
| Delete<br/>Backspace               | 删除选中的节点                                                                                              |
| Ctrl + Delete<br/>Ctrl + Backspace | 删除当前图形                                                                                           |
| Space                              | 当按下并移动光标时，移动画布                                                             |
| Ctrl + Click<br/>Shift + Click     | 将点击的节点添加到选择中                                                                                      |
| Ctrl + C/Ctrl + V                  | 复制并粘贴选中的节点（不保留未选中节点的输出连接）                     |
| Ctrl + C/Ctrl + Shift + V          | 复制并粘贴选中的节点（保留未选中节点的输出连接到粘贴节点的输入） |
| Shift + Drag                       | 同时移动多个选中的节点                                                                      |
| Ctrl + D                           | 加载默认图形                                                                                                 |
| Q                                  | 切换队列的可见性                                                                                     |
| H                                  | 切换历史的可见性                                                                                       |
| R                                  | 刷新图形                                                                                                      |
| Double-Click LMB                   | 快速搜索要添加的节点                                                                                      |
</Tab>
